Smart Home Hub & Automation Platform

Technical Specification

This deployment establishes a central IoT gateway utilizing an ESP32 dual-core microcontroller acting as the main bus manager. Hardware scheduling is handled via FreeRTOS threads dividing sensor queries and connectivity requests.

  • MCU: ESP32-WROOM-32D

Robotics & Kinematics

PID Controlled Autonomous Robot

Technical Specification

An autonomous line tracking vehicle utilizing closed-loop feedback design. The mathematical core models track deviance using raw analog values from infrared receptors, converting errors through proportional, integral, and derivative equations to throttle two high-speed DC gear motors.

Control Code Architecture

Firmware operates a high-frequency polling interrupt. Track values are weighted, and the resulting delta manages speed via pulse-width modulation (PWM) pins on the Atmega328P.

Solar & Low-Power Nodes

Low-Power Solar Weather Station

Technical Specification

Outdoor monitoring system configured for long-term survival under solar-cell backup power. Sleep cycles power down the central ESP8266 radio chip completely. Memory preservation is handled inside RTC memory registers, retaining previous calibration parameters during cold reboots.

Power Consumption Metrics

  • Sleep Current: ~15µA
  • Active State (Sensor Read): 15mA (duration: 350ms)
  • RF Transmission: 120mA (duration: 120ms)
